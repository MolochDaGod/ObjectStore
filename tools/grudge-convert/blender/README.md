# Blender MCP addon

Run from ObjectStore:

```bash
npm run blender:mcp:install-addon
```

This downloads [ahujasid/blender-mcp](https://github.com/ahujasid/blender-mcp) `addon.py` here.

Install in Blender → enable **Blender MCP** → Connect on port **9876**.

Production bake remains: `npm run convert -- …`
